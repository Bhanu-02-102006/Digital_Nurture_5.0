# Git Hands-On Lab

This folder contains the Git Hands-On Lab completed as part of the Digital Nurture 5.0 program.

## Objectives

- Initialize a local Git repository
- Track files using Git
- Stage and commit changes
- Verify repository status
- Push changes to GitHub

## Files

- `welcome.txt` – Sample text file added to demonstrate Git version control.

## Git Commands Practiced

```bash
git init
git status
git add
git commit
git push
git pull
```

## Outcome

Successfully created a Git repository, tracked files, committed changes, and pushed the project to GitHub.
