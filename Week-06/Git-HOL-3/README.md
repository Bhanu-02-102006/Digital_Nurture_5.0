# Git Branching and Merging Hands-On Lab

This exercise demonstrates Git branching and merging.

## Objectives

- Create a new branch
- Switch between branches
- Add and commit changes
- Merge the branch into the main branch
- View commit history
- Delete the merged branch

## Branch Used

- GitNewBranch

## Outcome

Successfully created a branch, committed changes, merged it into the main branch, viewed the commit history, and deleted the branch.
