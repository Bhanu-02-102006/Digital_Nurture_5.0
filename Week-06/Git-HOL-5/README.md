# Git Cleanup and Push to Remote Hands-On Lab

This exercise demonstrates how to synchronize the local Git repository with the remote repository.

## Objectives

- Verify the repository is in a clean state
- List all available branches
- Pull the latest changes from the remote repository
- Push pending changes to the remote repository
- Verify that changes are reflected on GitHub

## Git Commands Practiced

```bash
git status
git branch
git branch -a
git pull origin main
git push origin main
```

## Outcome

Successfully verified the repository status, synchronized the local repository with the remote repository, and confirmed that all changes were available on GitHub.
